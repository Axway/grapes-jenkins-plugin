If you check this option, Jenkins Grapes plugin will post build information to Grapes at the end of the build. It will add information about the SCM information (scm-repo, scm-url, scm-revision), the Jenkins builder (jenkins-host, jenkins-node) and the build (maven-version, maven-goal, build-date, java-version). You will be able to retrieve this information using Grapes @ /module/{name}/{version}/buildinfo.
